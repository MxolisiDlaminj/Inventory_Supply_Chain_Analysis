# Inventory & Supply Chain SQL Analysis Project

## Project Overview

This project focuses on analysing inventory and supply chain operations using SQL. The aim of the project was to extract meaningful business insights from structured datasets related to inventory management, sales performance, supplier efficiency, profitability, and warehouse operations.

The project demonstrates practical SQL skills used in data analysis and business intelligence environments.

---

# Objectives

* Analyse inventory stock levels and reorder patterns
* Identify top-performing and underperforming products
* Evaluate supplier delivery performance
* Measure profitability and revenue trends
* Analyse warehouse operational efficiency
* Support business decision-making using data insights

---

# Tools & Technologies

* SQL
* Relational Database Concepts
* Data Analysis Techniques

---

# SQL Concepts Applied

The project includes the use of:

* SELECT statements
* WHERE clauses
* GROUP BY and ORDER BY
* Aggregate functions (SUM, AVG, COUNT)
* HAVING clauses
* Subqueries
* Views
* Window functions (RANK)
* Data filtering and sorting

---

# Key Business Questions Answered

## Inventory Management

* Which products frequently run low on stock?
* Which warehouses hold the highest inventory?
* What are the average stock levels per category?

## Sales & Demand Analysis

* Which products generate the highest sales revenue?
* Which categories have the highest demand?
* Which products are underperforming?

## Supplier Performance

* Which suppliers have the longest delivery times?
* Which suppliers contribute to stock shortages?

## Profitability Analysis

* Which products generate the most profit?
* What is the average profit margin per category?

## Operational Efficiency

* Which warehouse processes the highest sales volume?
* Which products remain in inventory the longest?
